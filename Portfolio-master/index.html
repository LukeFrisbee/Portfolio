About Me

Hello! When I am not studying or tutoring Computer Science at Concordia University Irvine I like to spend my time working on personal software projects. I have a huge passion for not only developing video games but programming pretty much anything in general! Unity Engine, Visual Studio, and C# are my go-to tools for working on projects, although I love working with many others such as C++, Java, and Eclipse. Projects

3D RogueLike

This 3D Roguelike designed specifically for mobile phones is a project I’ve been constantly working on for about 4 months. It incorporates many new programming practices such as Data-Oriented programming using Unity Engine’s new “Data Oriented Technical Stack” also known as DOTS. The project has been built solely by me using Unity Engine, Visual Studio, and primarily the C# language. My three favorite functions I’ve programmed would be the joystick movement/combat system, the individual AI agents, and the automatic targeting system. However what I’m most proud of so far is the code organization and optimization throughout the project, as it currently contains 40+ individual script files, many of which inherit and implement one another. This project exemplifies some of my best work! (See Video)

StarComet

Star Comet is a mobile game I released during my sophomore year of high school. Being my first ever release, this simple vertical scrolling game is one that I take a lot of pride in. Star Comet is where my love for game development exponentially grew, as I learned and executed the entire process incorporated in making video games, whether it was code, art, or publishing. This project not only increased my programming and other game-oriented skills, but it also taught me a great deal of patience and perseverance. Without those two character traits, the completion of this project or any future ones would have never been possible. (See Video)
