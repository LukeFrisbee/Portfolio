TDS

TDS is 4 player online co-op base builder, utilizing Photon Fusion as its network solution. A frankenstien of many genres, it primarily combines elements from twin-stick shooters, roguelikes, and RTS games. TDS is currently very early in development and is planned to release on PC and Console in the distant future.

Space Sim

This arcade space simulator is a explorative project of mine. The genre was chosen to challenge my technical skills and my grasp of in-depth game engine systems, such as physics and rendering. I heavily enjoy developing the art; it keeps my modeling skills at a low scope, forgoing complex animation and allowing me to focus on the game itself.

3D Battleship

3D computer implementation of the strategy game Battleship developed as a framework for software developers to compete in a C# algorithm tournament. Each competitor’s code is compiled into a DLL and runs through the project. These algorithms are then pitted against one another, and the project calculates and analyzes each competitor’s AI’s decision.

3D RogueLike

As a prototype for a mobile 3D RogueLike game idea, this project was developed throughout my first year at university. The project, along with many others, encouraged exercising self-growth alongside my courses, as I wanted to challenge my code writing capabilities and expand my understanding of what makes video games tick.

Star Comet

Star Comet is a simple, traditional scrolling game designed for android and ios. As the first game I ever developed (eight years ago), it holds a nostalgic fondness and place for me. This humble mobile game dawned a lifelong passion for computer science and video game development.
