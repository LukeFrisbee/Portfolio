3D Battleship

3D computer implementation of the strategy game Battleship developed to allow software developers to compete in a C# algorithm tournament. Each competitor’s code is compiled into a DLL and runs through the project. These algorithms are then pitted against one another, in which the project calculates and error checks each battleship AI’s decision.

Space Sim

This arcade space simulator is my current primary project. I choose this genre to challenge my technical skills and my grasp of in-depth game engine systems, such as physics and rendering. I also heavily enjoy developing the art, as it keeps my modelling skills at a low scope, forgoing animation and allowing me to focus on the game itself.

3D RogueLike

This project was a prototype to a mobile 3D RogueLike game idea I had been developing in my head throughout university. I used the project along many others as a exercise in self-growth alongside my courses, as I wanted to challenge my code writing capabilities and expand my understanding of what makes video games truly tick.

Star Comet

Star Comet is a simple, traditional scrolling game designed for andriod and ios. This project holds a special place in my heart as it was the first video game I ever developed, about seven years ago. This humble mobile game was the dawn to a life long passion of computer science and video game development.
