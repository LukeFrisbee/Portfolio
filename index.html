Projects

3D RogueLike

This 3D Roguelike designed specifically for mobile phones is a project I’ve been constantly working on for about 4 months. It incorporates many new programming practices such as Data-Oriented programming using Unity Engine’s new “Data Oriented Technical Stack” also known as DOTS. The project has been built solely by me using Unity Engine, Visual Studio, and primarily the C# language. My three favorite functions I’ve programmed would be the joystick movement/combat system, the individual AI agents, and the automatic targeting system. However what I’m most proud of so far is the code organization and optimization throughout the project, as it currently contains 40+ individual script files, many of which inherit and implement one another. This project exemplifies some of my best work! (See Video)

Entropic Space

Entropic Space is a recent five-day project built for a Game Jam. The theme of the Jam was "Lost in Space." I took it pretty literally, creating a minimalistic 3D space shooter. While it is a simple game, there were specific mechanics I found challenging to implement, especially within the five-day time frame I had. Most notably, the Flight System and Enemy Ship AI posed some obstacles. Luckily I had a team of friends to help me out: one with development, one with 3D modeling, and finally one with composing sounds and music. I lead them through the process, and due to their help, I was able to focus my attention on the more complicated obstacles. Thanks to them, I was able to complete this project within the given time-frame. (For more information, feel free to visit the itch.io widget posted below )

Soul Snatcher

Soul Snatcher is a WebGL game built within 48 hours for a Game Jam. The theme required in the competition was "Death is Good." To go by the theme, the player does not lose when they die. Instead, they power up, attacking more quickly. The catch is every time the player shoots; they lose some of their "soul" life. Run out of life, and it's game over! The player can't die too many times, or else they won't be able to handle the power given to them.
